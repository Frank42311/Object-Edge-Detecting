OpenCV Algorithm from Scratch

Noise Reduction: Mean Filter; Median Filter; Gaussian Filter

Edge Detection: Canny Edge Detector; Marr Hildreth Edge Detector; Sobel Edge Detector; Laplacian Edge Detector
